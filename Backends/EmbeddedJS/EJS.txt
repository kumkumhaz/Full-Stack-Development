Embedded JavaScript is a templating language for JavaScript, It allows us to embed JavaScript code directly within the HTML.
It's commonly used with Node.js and Express.js for server-side rendering.


EJS Tags

<%= variable %>              JS Output
<% console.log("hello") %>   JS Execute
<%- <h1> Hello </h1> %>      Render HTML
<%% %%>  => <%% <% %%> or <%% %> %%>  Show <% or %> => whatever we write in between the <%% %%> , that will be pretend as sign or a text but not a EJS Tags
<%# This is a comment %>     Stop Execution
<%- include("<FILE NAME>") %>  =>  <%- include("header.ejs) %> => insert another EJS FILE
