Definition: Object Oriented Programming is a style of programming that is associated with the concept of Class, Objects, and various other means like Inheritance, Polymorphism, Encapsulation, and Abstraction. nitially, when we did not have Object Oriented Programming, we used to have Procedural Programming at that time like in the C language. By Procedural Programming, we only mean that we used procedures or functions or we can say that the entire program is based upon functions and all that which perform various operations on data. But when C++ came with the concept of OOPs in which everything is represented in terms of Objects and Classes which contain both data functions. Objects and classes help us to manage the data flows between functions or methods in a program.
